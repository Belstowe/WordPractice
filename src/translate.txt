/ Программа игнорирует всё после слэша до новой строки.
/ Программа также игнорирует пробелы.
/ Однако если будет необходим пробел внутри заданий, используется подчёркивание (_). Внутри программы в строке он автоматически превращается в пробел.
/ : - программа начинает читать слова на зарубежном.
/ | - "или", используется как разделитель синонимов/разных вариантов перевода. Вносится в строку в самой программе и разделяет субстроки, из которых программа затем будет выбирать случайный.
/ = - чтение иноязычных слов заканчивается, переход на русский.
/ ; - "пара" перевода заканчивается.
/ . - программа прекращает читать файл.

/ Пример:
: illuminate = излучать;
: test = тест | испытание | испытать | тестировать;
: solid|hard = твёрдый|твердый;
: bite_the_bullet = стиснуть_зубы.

/ Это в программе будет выглядеть так:
/ word[0].first = "illuminate";
/ word[0].second = "излучать";
/ word[1].first = "test";
/ word[1].second = "тест|испытание|испытать|тестировать";
/ word[2].first = "solid|hard";
/ word[2].second = "твёрдый/твердый";
/ word[3].first = "bite the bullet";
/ word[3].second = "стиснуть зубы";
